# 🧑‍💻 Developer Portfolio

A clean, responsive developer portfolio built with **React** and **Tailwind CSS**.

## 🚀 Live Demo

[View Live](https://rameez-portfolio-five.vercel.app/)

## ⚙️ Tech Stack

- React.js
- Tailwind CSS
- Framer Motion
- Vite (or CRA)

## 📂 Features

- Experience timeline
- Projects showcase
- Career stats
- Responsive design

## 🛠 Setup

```bash
git clone https://github.com/your-username/portfolio.git
cd portfolio
npm install
npm run dev
